Типография

Шрифты

В проекте обязательно должны присутствовать три типа основного шрифта:

  • Обычный (regular)
  • Жирный (bold)
  • Курсивный (italic)

Настройка

SASS

Настройкой является переменная $headings, значением которой является список размеров заголовков. 

$headings:
   (h1 24px 36px 48px),
   (h2 20px 30px 40px),
   (h3 19px 25px 31px),
   (h4 18px 21px 25px),
   (h5 17px 18px 20px),
   (h6 16px 16px 16px);

Цитата

Пример

<blockquote>
  Ты не несешь ответственности за то, чего ждут от тебя другие люди. Если от тебя ждут слишком многого,
  то это их ошибка, а не твоя вина.
  <cite>Ричард Фейнман</cite>
</blockquote>
Ты не несешь ответственности за то, чего ждут от тебя другие люди. Если от тебя ждут слишком многого, то это их ошибка, а не твоя вина. Ричард Фейнман

Small

Описание

В HTML5 тег <small> теперь имеет немного иной смысл — «Текст мелким шрифтом» (информация, являющаяся юридической формальностью, как-то лицензия предприятия или юридический адрес и др.)

Стили применяются к тегу <small>

Пример

<small>Текст</small>

SASS

Параметры

Имя Описание Тип Зн. По-умолчанию
$small-font-size Размер текста number px #$base-font-size — 2

Ссылка

Пример

<a href="#"> Ссылка </a>
Ссылка

SASS

Параметры

Имя Описание Тип Зн. По-умолчанию
$link-color Цвет color - #0000FF
$link-color-hover Цвет color - lighten (#0000ff, 5%)
$link-color-active Цвет: active ссылки color - darken ($link-color, 5%)
$link-text-decoration Подчеркивание ссылки string normal | invert | reset normal

Подчёркивание у ссылки может быть в трёх состояниях

Значение Обычное состояние Наведенное состояние
normal подчеркивание нету подчеркнуто
invert подчеркнуто подчеркивание нету
reset подчеркивание нету подчеркивание нету


Списки

Так обычно списки используются для навигации и прочего, по-умолчанию стили списков «сброшены» (reset). Чтобы вернуть нормальную работу списков, надо родителю списков добавить класс .b-typo-reset 

<div class="b-typo-reset"> Ссылка
      <ul>
          <li> ... <li>
      <ul>
      <ol>
          <li> ... <li>
      <ol>
</div>

Несортированный список

Стили применяются к <ul>, если у родителя есть класс .b-typo-reset.

  <ul>
      <li> Пункт списка <li>
  <ul>
  • Пункт списка
  • Пункт списка
  • Пункт списка
    • Подпункт списка
    • Подпункт списка
    • Подпункт списка

Сортированный список

Стили применяются к <ol>, если у родителя есть класс .b-typo-reset.

  <ol>
      <li> Пункт списка <li>
  <ol>
  1. Пункт списка
  2. Пункт списка
  3. Пункт списка

    2. Подпункт списка
    3. Подпункт списка

$list-sublist-padding-left Отступ слева (список в списке) number gutter 4
