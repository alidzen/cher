Формы

Текстовые поля

Использование

Стили применяются к стандартным текстовым инпутам. Посмотреть можно в block/_inputs.jade

jade

  input(type="text")
  input(type="password")
  input(type="tel")
  input(type="email")
  input(type="search")
  input(type="url")

Параметры текстовых полей

Имя Описание Тип Зн. По-умолчанию
$input-color Цвет текста color - $black
$input-bg-color Цвет фона color - $white
$input-bd-width Размер бордера number px 1px
$input-bd-color Цвет бордера color - $black
$input-border-radius Радиус скругления number px 0
$input-font-size Размер текста number px 16px
$input-font-family Шрифт string - $font
$input-height Высота кнопки number rhythm 2
$textarea-min-height Минимальная высота textarea number rhythm 5

Дополнительные параметры — focus

Имя Описание Тип Зн. По-умолчанию
$input-bd-color-focus Цвет бордера color - #ffa500

Дополнительные параметры — placeholder

Имя Описание Тип Зн. По-умолчанию
$input-placeholder-color Цвет подсказки color - #ccc
$input-placeholder-font-style Начертание шрифта string normal | italic | oblique | inherit italic
$input-placeholder-font-size Размер шрифта number px $h5
$input-placeholder-opacity Прозрачность текста number 0.1 1

Анимированный label

JADE

Миксин

+anim-label('name', 'type', required)

Параметры

Имя Описание Тип Зн. По-умолчанию
name Текст в label string - -
type Тип поля string - text
required Тип поля bool true|false false

SASS

Параметры

Имя Описание Тип Зн. По-умолчанию
$anim-label-font-size Размер текста number px $input-font-size
$anim-label-font-size-focus Размер текста при фокусе поля number px 10px
$anim-label-tt-up Преобразование текста в прописные или заглавные bool false/true false
$anim-label-tt-up-focus Преобразование текста в прописные или заглавные при фокусе поля bool false/true true
$anim-label-color Цвет текста лэйбла color - inherit
$anim-label-color-focus Цвет текста лэйбла при фокусе поля color - inherit

Чекбоксы

Описание

Структуру можно в block/_checkbox.jade

JADE

  .b-form__checkbox
      input(id="ch1" type="checkbox")
      label(for="ch1") Здесь может быть текст

В лейбле учтена многострочность.

Параметры чекбоксов

У чекбокса четыре состояния. Все картинки нужно положить в папку /sprites/1x, а в папку /sprites/2x их увеличенные в два раза версии.

Имя Описание Тип Зн. Название файла
$checkbox Невыбранный checkbox string - checkbox.png
$checkbox-checked Выбранный checkbox string - checkbox-checked.png
$checkbox-hover Наведённый checkbox string - checkbox-hover.png
$checkbox-disabled Выключенный checkbox string - checkbox-disabled.png

Селект

Использование

Для работы используется библиотке selectric. Документация

Для правильной работы нужно подключить стили библиотеки.

Стили применяются к обычному <select>

JADE

select
  option(value="1") Пункт 1
  option(value="2") Пункт 2

HTML

  <select>
      <option value="1">пунк 1</option>
      <option value="2">пунк 2</option>
  </select>

SASS

Параметры

Имя Описание Тип Зн. По-умолчанию
$select-font-size Размер текста px px 16px
$select-font-family Шрифт string - $font
$select-color Цвет color - $black
$select-border-width Размер бордера number px 1px
$select-border-color Цвет бордера color - #ccc
$select-border-radius Радиус скругления number px 4px
$select-height Высота селекта number rhytm $btn-height

Параметры — пункта селекта

Имя Описание Тип Зн. По-умолчанию
$select-item-background-color Цвет фона (select и option) color - $white
$select-item-background-color-selected Цвет фона выбранного пункта color - #ccc
$select-item-background-color-hover Цвет фона пункта при наведении color - lighten ($select-item-background-color-selected, 15%)
$select-item-color-hover Цвет выбранного пункта и при наведении color - #999
$select-item-border-width Размер бордера у пункта number px 1px
$select-item-border-color Цвет бордера у пункта color - #eee
