Логотип Кельник

Описание

Логотип сделан в SVG, поэтому может принимать любую ширину и высоту.

Добавление в проект

Jade

  include blocks/_kelnik

Параметры

Имя Описание Тип Зн. По-умолчанию
$kelnik-color Цвет иконки color - $black
$kelnik-opacity Прозрачность иконки number 0.1 1
$kelnik-width Ширина логотипа number px 124px
$kelnik-height Высота логотипа number px 36px

Дополнительные параметры — hover effect

Имя Описание Тип Зн. По-умолчанию
$kelnik-color-hover Цвет иконки color - #808080
$kelnik-opacity-hover Прозрачность иконки number 0.1 1
