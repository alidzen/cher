Как правильно пользоваться сеткой

Все настройки находятся в style/base/_grid.scss

Параметры

Имя Описание Тип Зн. По-умолчанию
$columns Количество колонок number - 12
$column-width Ширина колонки number px 60px
$gutter-width Расстояние между колонками number px 20px

Функции

Имя Описание Тип Зн. По-умолчанию
span () Ширина колонки. Принимает второй параметр «из скольки» number px 60px
gutter () Межколоночное пространство number - 1

Миксины

Имя Описание Тип Зн. По-умолчанию
container Максимальная ширина контейнера сетки number px | % 960px
span Ширина колонки number - 1
gutter Расстояние между колонками number - 1
push Отталкивает колонку слева number - 1
post Отталкивает колонку справа number - 1
prefix Внутренний отступ слева number - 1
suffix Внутренний отступ справа number - 1
first Колонка позиционируется по порядку - - -
last Колонка позиционируется в противоположном порядке - - -
gallery Помогает расположить колонки с разной высотой number - 1
grid-clearfix Очищает поток - - -

$column-width

$column-width — ширина колонки. Т.к. сетка у нас резиновая, а не статичная, ширина колонки высчитывается в процентном соотношеннии.

Какую бы мы не указали ширину колонки 60px или 70px, она всегда будет равняться 8.33%, , но увеличится контейнер сетки, о нём чуть позже.

Функции

span

span — функция принимает количество колонок и возвращает ширину этих колонок.

  width: span(6); // 50%;

span принимает второй параметр «из скольки». Нужен, когда колонки располагаются внутри колонок.

  width: span(6 of 10); // 60%;
  width: span(6 of 12); // 50%;

gutter

gutter() — функция принимает количество гаттеров и возврашает ширину этих гаттеров.

  padding-left: gutter(6); // 60px;

Также стоит запомнить gutter(1) равняется половине $gutter-width. Т.к. желоба располагаются по обе стороны колонки.

Миксины

container

Основная задача контейнера это ограничивать сетку.

Если ничего не передават миксину, то для вычисления ширины контейнера используются переменные $columns, $column-width, $gutter-width

Миксин принимает любую величину (px или %).

  @include container(); // max-width: 960px; - велечина высчитывается из основных настроек
  @include container(1440px); // max-width: 1440px;
  @include container(90%); // max-width: 90%;

span

Миксин span строит колонки. Миксин принимает несколько разных параметров

of — 1 of 10 — добавляет контекст «из скольки» колонок. Обычно требуется, когда колонки вложены в колонку.
last — необязательный параметр, говорящий колонке строится с конца.
first — обратный аналог предыдщему параметру.
center — центрирует колонку.
full — сокращение от span (12). Говорит колонкам занимать всё доступное пространство.
half — сокращение от span (6). Говорит колонкам занимать половину доступного пространства. 

  @include span(10); // занимай 10 колонок
  @include span(2 of 10); // занимай 2 колонки из 10
  @include span(10 center); // занимай 10 колонок и центрируйся
  @include span(2 of 10 center); //занимай 2 колонки из 10 и центрируйся
  @include span(2 last); // занимай 2 колонки и встань вконец

gutter

Устанавливает элементу внтуренние отступы по бокам равные желобам.

  @include gutter(1); // padding-left: 10px; padding-right: 10px;

push

Устанавливает отступ слева, равный количеству передаваемых колонок.

  @include push(1); // margin-left: 8.33%;

post

Устанавливает отступ справа, равный количеству передаваемых колонок.

  @include post(1); // margin-right: 8.33%;

prefix

Устанавливает внутренний отступ слева, равный количеству передаваемых колонок.

  @include prefix(1); // padding-left: 8.33%;

suffix

Устанавливает внутренний отступ справа, равный количеству передаваемых колонок.

  @include suffix(1); // padding-right: 8.33%;

last

Указывает колонке быть последней.

  @include last(); // float: right;

first

Указывает колонке идти в потоке. Т.е. в обычном её состоянии.

  @include first(); // float: left;

gallery

Ведёт себя как @include span(); с одним бонусом, не нужно учитывать высоту элементов.

  @include gallery(6);

grid-clearfix

Обычный clearfix, просто очищает поток.

  @include grid-clearfix();
