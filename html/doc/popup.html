Попап

В проекте в качестве попапа используется Magnific. Документация

Описание

Контентом в попапе может быть всё что угодно — карта, галерея, картинка, форма, текст. Весь контент помещайте в .b-popup.

Использование

  <a href="#popup" class="j-popup">Открыть попап<a>
  <div id="#popup">
      ...
  <div>

В данном примере ссылка покажет в попапе, всё что находится в #popup.

Параметры

Параметры применяются через data-атрибуты на ссылке

Параметры — основные

Имя Описание Тип Зн. По-умолчанию
data-type Тип попапа string inline | iframe | image inline

Пример 
  <a href="#" data-type="inline">Открыть попап<a>

Параметры — темы

Имя Описание Тип Зн. По-умолчанию
data-theme На родителя попапа вешается модификатор string - -

Пример

  <a href="#" data-theme="orange">Открыть попап<a>

На основе этого data-атрибута создаётся класс b-theme_theme_orange. И вы можете унаследовшись от него задать свои стили.

SASS

.b-theme_theme_orange {
      .b-popup {
          ...
      }
}

Параметры — полноэкранный

Имя Описание
data-fullsize Полноэкранный попап

Пример 
  <a href="#" data-fullsize>Открыть попап<a>

Параметры — анимация исчезания и появления

Плавное появление и скрытие попапа

Имя Описание Тип Зн. По-умолчанию
data-toggle-time Время появления и исчезновения number ms 0

Пример

  <a href="#" data-toggle-time="300">Открыть попап<a>

Параметры — тело попапа

SASS

Имя Описание Тип Зн. По-умолчанию
$popup-color Цвет текста color - $base-font-color
$popup-bg-color Цвет фона color - $white
$popup-overlay-color Цвет оверлея color - $black
$popup-overlay-opacity Прозрачность оверлея number .10 .8
$popup-border-radius Радиус скругления контента попапа number px 0

Параметры — кнопка

Кнопка в попапе имеет два вида — кнопка в мобильном виде и крестик на десктопе.

В мобильной версии все параметры по-умолчанию принимаются от .b-btn

Для отображения крестика добавить в папки /sprites/1x и /sprites/2x картинки popup-close.png.

Кнопка мобильная

SASS

Имя Описание Тип Зн. По-умолчанию
$popup-btn-mobile-color Цвет текста color - $btn-color
$popup-btn-mobile-bg-color Цвет фона color - $btn-bg-color
$popup-btn-mobile-font-size Размер шрифта number px $btn-font-size
$popup-btn-mobile-line-height Межстрочный интервал number px $base-line-height
$popup-btn-mobile-font-family Шрифт string - $btn-font-family
$popop-btn-mobile-height Высота кнопки number rhythm $btn-height
$popup-btn-mobile-tt-up Преобразование текста в прописные или заглавные bool false|true $btn-tt-up / false

Кнопка десктопная

SASS

Имя Описание Тип Зн. По-умолчанию
$popup-btn-close-image Название файла картинки крестика string - popup-close
$popup-btn-opacity Прозрачность крестика number .1 .5
